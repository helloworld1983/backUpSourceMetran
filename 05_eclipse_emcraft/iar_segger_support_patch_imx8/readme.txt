  About this patch
   =========================================
   JLink_V633d.zip provides i.MX8 series Cortex-M4 cores support in JLink V633d.
   arm.zip enables i.MX8 series Cortex-M4 cores debugging in IAR v8.22.1. 


   MD5SUM:
     ac8b932d94e2ca5c1717b45535ce2eab   arm.zip
     cc6400e1d6e9d91ce5ab5e2c9ff2a4b9   JLink_V633d.zip
   

   How to use this patch
   =========================================
   1.arm.zip

    Please unzip the arm.zip and override all the existing files in the arm folder in your IAR installer path. 
    For example, C:\Program Files (x86)\IAR Systems\Embedded Workbench 8.0\arm.
   
   2.JLink_V633d.zip
     a.Please install SEGGER software, which can be downloaded from www.segger.com/jlink-software.html.

     b.Please unzip the JLink_V633d.zip and override all the existing files in the JLink folder in your SEGGER installer path. 
    For example, C:\Program Files (x86)\SEGGER\JLink_V633d.
